# 1. Add two strings

## Problem
given two binary strings return their sum (also a string)

## Examples
### Example 1
#### input
```text
num1: '1'
num2: '2'
```
#### output
```text
'3'
```

### Example 2
#### input
```text
num1: '9'
num2: '1'
```
#### output
```text
'10'
```

## Constraints
* Do not convert whole string to integer and calculate it. To be specific, this code is not allowed
```python
def sol(num1, num2):
    return str(int(num1) + int(num2))
```
